n8n Integration

In n8n, use an HTTP Request node:

Method: POST
URL: https://pdf.luxeillum.com/api/documents
Header: x-api-key: pfk_your_key_here
Body (JSON):
{
"template_id": "your-template-slug",
"data": {"{{ $json }}"}
}

Response includes url — a direct link to the generated PDF.
